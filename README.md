# -
              这是一个基于Django框架开发的电商平台项目（shopweb），采用前后台分离架构。项目主要分为myadmin后台管理和web前台用户两个应用模块。技术栈包括Django 3.2.5、MySQL数据库、HTML/CSS/JavaScript等。核心功能涵盖商品管理（商品增删改查、分类管理）、用户系统（注册登录、个人中心）、交易管理（购物车、订单处理）及收藏夹功能。数据模型主要包括User（普通用户）、Admin（管理员）、Goods（商品）等。项目结构清晰，通过Django的MVT模式实现业务逻辑，支持图片上传、Session认证等功能，并计划添加管理员向用户发送消息及评语的功能。
